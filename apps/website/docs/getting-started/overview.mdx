# Overview

## Introduction

Tailwind-Joy is an open-source React component library that styles the components of Joy UI (an alternative to Material UI) in the Tailwind CSS way.

## Why use Tailwind-Joy

Visually equivalent to Joy UI (except for the `font-family`[^1]), but does not depend on Emotion. Of course, it does not depend on styled-components either.

[^1]: Tailwind-Joy does not use any particular font as its default font. The default font can be selected to suit the end user's preference.

For example, comparing the button components:

export function GridItem({ children }) {
  return (
    <div className="border-joy-neutral-300 dark:border-joy-neutral-500 flex-1 rounded-lg border border-solid p-4">
      {children}
    </div>
  );
}
