# Button Group

<AvailableFrom version="0.2.0" />

The Button Group combines a set of related buttons.

## Basics

```jsx
import { ButtonGroup } from 'tailwind-joy/components';
```

The Button Group component can wrap [`Button`](./button) and [`IconButton`](./button#icon-button).

import { MdSettings } from 'react-icons/md';

export function ButtonGroupBasics() {
  return (
    <DisplayStand>
      <ButtonGroup>
        <Button>One</Button>
        <Button>Two</Button>
        <Button>Three</Button>
        <IconButton>
          <IconAdapter>
            <MdSettings />
          </IconAdapter>
        </IconButton>
      </ButtonGroup>
    </DisplayStand>
  );
}

## Customization

### Variants

The Button Group component supports four variants: `solid`, `soft`, `outlined` (default), and `plain`.

The `variant` prop is passed to the buttons, not the Button Group itself.

export function ButtonGroupVariants() {
  return (
    <DisplayStand className="flex-col">
      <ButtonGroup variant="solid">
        <Button>One</Button>
        <Button>Two</Button>
        <Button>Three</Button>
      </ButtonGroup>
      <ButtonGroup variant="soft">
        <Button>One</Button>
        <Button>Two</Button>
        <Button>Three</Button>
      </ButtonGroup>
      <ButtonGroup variant="outlined">
        <Button>One</Button>
        <Button>Two</Button>
        <Button>Three</Button>
      </ButtonGroup>
      <ButtonGroup variant="plain">

### Disabled

Use the `disabled` prop to disable all the buttons.

The `disabled` prop is passed to the buttons, not the Button Group itself.
Note that if you explicitly specify the `disabled` prop on the buttons directly, it will override what is used in the Button Group.

export function ButtonGroupDisabled() {
  return (
    <DisplayStand>
      <ButtonGroup disabled>
        <Button>One</Button>
        <Button>Two</Button>
        <Button>Three</Button>
        <IconButton disabled={false}>
          <IconAdapter>
            <MdSettings />
          </IconAdapter>
        </IconButton>
      </ButtonGroup>
    </DisplayStand>
  );
}

### Spacing

Use `spacing` prop to control the gap between buttons.
If the `spacing` is set to `0` (by default), the radius of the buttons will be adjusted to form a continuous surface.

export function ButtonGroupSpacing() {
  return (
    <DisplayStand>
      <ButtonGroup spacing="0.5rem">
        <Button>One</Button>
        <Button disabled>Two</Button>
        <Button>Three</Button>
        <IconButton>
          <IconAdapter>
            <MdSettings />
          </IconAdapter>
        </IconButton>
      </ButtonGroup>
    </DisplayStand>
  );

### Separator color

When the Button Group's variant is **not** `outlined`, separators are created between buttons.
To control the color of the separator, override the CSS variable `--ButtonGroup-separatorColor` via the `style` prop.

import { useState } from 'react';

export function ButtonGroupSeparatorColor() {
  const [hue, setHue] = useState(0);
  return (
    <DisplayStand className="flex-col">
      <div className="flex items-center gap-2">
        <span className="w-10 text-right">Hue:</span>
        <input
          type="range"
          name="hue-range"
          min="0"
          max="360"
          defaultValue="0"
          onChange={(e) => setHue(e.currentTarget.value)}
        />
        <span className="w-10 text-left">{hue}</span>
      </div>
      <ButtonGroup
        variant="soft"
        style={{ '--ButtonGroup-separatorColor': `hsl(${hue} 100% 50%)` }}
      >
        <Button>One</Button>
        <Button>Two</Button>
        <Button>Three</Button>
      </ButtonGroup>
    </DisplayStand>
  );
}

<ButtonGroupSeparatorColor />

## Anatomy

The Button Group component is composed of a single root `<div>` element that wraps around its contents:

```html
<div role="group" class="tj-button-group-root ...">
  <!-- Button Group contents -->
</div>
```
