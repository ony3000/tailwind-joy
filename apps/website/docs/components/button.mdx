# Button

Buttons let users take actions and make choices with a single tap.

## Basics

```jsx
import { Button } from 'tailwind-joy/components';
```

The Tailwind-Joy Button behaves similarly to the native HTML `<button>`, so it wraps around the text displayed on its surface.

The demo below shows the three basic states available to the Button: default, disabled, and loading.

export function ButtonBasics() {
  return (
    <div className="border-joy-neutral-300 dark:border-joy-neutral-500 flex items-center justify-center gap-x-4 rounded-lg border border-solid p-4">
      <Button className="font-[inherit]">Button</Button>
      <Button className="font-[inherit]" disabled>
        Disabled
      </Button>
      <Button className="font-[inherit]" loading>
        Loading
      </Button>
    </div>
  );
}

<ButtonBasics />

### Disabled

Use the `disabled` prop to disable interaction and focus:

export function ButtonDisabled() {
  return (
    <div className="border-joy-neutral-300 dark:border-joy-neutral-500 flex items-center justify-center gap-x-4 rounded-lg border border-solid p-4">
      <Button className="font-[inherit]" disabled variant="solid">
        Solid
      </Button>
      <Button className="font-[inherit]" disabled variant="soft">
        Soft
      </Button>
      <Button className="font-[inherit]" disabled variant="outlined">
        Outlined
      </Button>
      <Button className="font-[inherit]" disabled variant="plain">
        Plain
      </Button>
    </div>
  );
}

## Customization

### Variants

The Button component supports four variants: `solid` (default), `soft`, `outlined`, and `plain`.

export function ButtonVariants() {
  return (
    <div className="border-joy-neutral-300 dark:border-joy-neutral-500 flex items-center justify-center gap-x-4 rounded-lg border border-solid p-4">
      <Button className="font-[inherit]" variant="solid">
        Solid
      </Button>
      <Button className="font-[inherit]" variant="soft">
        Soft
      </Button>
      <Button className="font-[inherit]" variant="outlined">
        Outlined
      </Button>
      <Button className="font-[inherit]" variant="plain">
        Plain
      </Button>
    </div>
  );
}

import { Button } from 'tailwind-joy/components';
